Rampant Robot Runner

About

Rampant Robot Runner was a team project that was made during the Spring of 2019 as a member of Cal Poly Pomona's Game Design & Development club. In this game, you play as robot ducking and dodging through a procedurally generated obtsacle course. A second player plays as the trap master who drops obstacles and sets off traps in the scene to try and deter the robot. This game was influenced by the PvP style party games you would find in the Mario Party series.

Contribution and Obstacles Faced

My contributions to this game includes the initial conceptualization and design of the game as well as the design and implementation of the UI. I programmed the main menu and setting scenes for the game. I also programmed the initial countdown included at the start of the game as well the obstacle selection interface for the trapmaster.

The biggest obstacle I faced during this project was my attempt at making the main menu interface respond to different types of input and making it consistent across all the other menu scenes like the 'settings' scene. Another obstacle I faced was properly creating the user interface of the settings menu to actually push changes to game's settings in its builds. These settings included game volume, resolution, and graphics quality.
