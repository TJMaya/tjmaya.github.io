Seismic Shift

About

Seismic Shift was a team project that was made during the Global Game Jam at the UC Irvine site. The idea for the game is based off of the Global Game Jam’s topic of “waves”. It is a 3d infinite runner game where you try to survive an earthquake for as long as you can while avoiding fleeing cars, falling buildings, and the seismic waves created by the earthquake.

Contribution and Obstacles Faced

My role in this project was to create the road presets used in the procedurally generated system that one of my teammates constructed as well as create a system that randomly instantiates the obstacles in different positions and quantities on the road. I created the road presets in a way that the player can’t leave the play area by using the buildings and dense city setting to my advantage. For the obstacle instantiation, I set different spawn points that hold a number value. Then I wrote a script that used a randomizer to decide which points obstacles would spawn on. I also created an event system that detected when the player would hit one of the crashed car obstacles in order to stun the player, destroy the obstacle, and instantiate a fiery explosion.

The biggest obstacle I faced was getting the randomizer to work the way I wanted it to. When I first created it, it seemed that there wasn’t enough diversity in the instances of the obstacles being created. In order to fix this, I added more spawn points and adjusted the scale of possibilities that the randomizer could choose from.
