Ninja Star

About

Ninja Star was a team project that was made during the Global Game Jam. It is a 2D side-scrolling puzzle platformer featuring a ninja starfish. The goal of the game is to reach the objective of each level. In order to traverse the platforms, the starfish needs to cut off its’ arm to shoot the arm as a projectile, and then respawn a new body at the arm, proceeded by transmitting its’ soul to the new body.

Contribution and Obstacles Faced

My role in this project was to put together the animations and sounds in the game. I did this using Unity’s animation controller and other built in tools as well as a few C# scripts using Visual Studio. I also added sounds to the obstacle interactions. I also worked on level design and selected a few soundtracks that would fit the game’s theme.

The biggest obstacle I faced during this project was actually trying to compile everyone’s work into one version of the game through Unity Collab. At one point everyone was working on their own version of the game and once joined, things obviously broke. To fix, this the team and I had to discuss and work it all out together on one version.
